# Contrib General — Sample Data Guide

This folder holds 10-row CSV samples from WRDS's `contrib_general` library —
a collection of curated research datasets contributed by academics and
practitioners. Each CSV mirrors one live database table.

The 20 tables are documented across two sub-guides:

| Guide | What it covers | Tables |
|---|---|---|
| [GUIDE_contrib_general_governance.md](GUIDE_contrib_general_governance.md) | Corporate governance events, ownership, text analysis, identifier crosswalks | ceo_turnover, classified_boards, blocksv, common_own_firm, common_own_industry, arc, as_firm_risks, cik_types, dealscan_link_bs, dealscan_link_comp, _states_ |
| [GUIDE_contrib_general_quant.md](GUIDE_contrib_general_quant.md) | Asset pricing factors, market beta, investor value, tax rates, real-economy data | factors, factors_daily, factors_eq, factors_eq_usd, factors_usd, better_beta, liva, marginal_tax, shale |

**Generated by:** `uv run python -m library_samples.export_small_samples`

**Row limit:** each CSV is `SELECT * FROM <table> LIMIT 10`.
