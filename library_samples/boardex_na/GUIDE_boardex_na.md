# BoardEx NA Guide

## Part 1: Conceptual Explanation

This folder stores the tiny BoardEx North America samples. The folder now
contains one 10-row CSV per live table in `boardex_na`, so quick inspection does
not miss tables or columns.

Artifact details:

- File pattern: `*.csv`
- Source: `library_samples/export_small_samples.py`
- Coverage: 39 live WRDS tables in `boardex_na`
- Row rule: each CSV contains `limit 10` rows
- Column rule: each CSV uses `select *`, so it contains all columns from that
  table
