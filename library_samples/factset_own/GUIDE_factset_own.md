# FactSet Own Guide

## Part 1: Conceptual Explanation

This folder stores the tiny FactSet ownership samples. The folder now contains
one 10-row CSV per live table in `factset_own`.

Artifact details:

- File pattern: `*.csv`
- Source: `library_samples/export_small_samples.py`
- Coverage: 34 live WRDS tables in `factset_own`
- Row rule: each CSV contains `limit 10` rows
- Column rule: each CSV uses `select *`, so it contains all columns from that
  table
