# TR Common Guide

## Part 1: Conceptual Explanation

This folder stores the tiny Refinitiv common-reference samples. The folder now
contains one 10-row CSV per live table in `tr_common`.

Artifact details:

- File pattern: `*.csv`
- Source: `library_samples/export_small_samples.py`
- Coverage: 30 live WRDS tables in `tr_common`
- Row rule: each CSV contains `limit 10` rows
- Column rule: each CSV uses `select *`, so it contains all columns from that
  table
