# AI Agent Knowledge Index

The following are key files that provide essential information and documentation for this workspace. When you need information about the WRDS datasets, schema, or sample data, consult these resources first:

- `outputs/postgres_libraries.csv`: Lists available data libraries in the WRDS PostgreSQL database.
- `outputs/postgres_tables.csv`: Lists tables available within each library.
- `outputs/postgres_columns.csv`: Details the column schemas across the WRDS PostgreSQL tables.
- `library_samples/`: Directory containing representative data samples from the datasets.
- `outputs/guide_numeric_libraries.md`: Developer guide detailing numeric libraries and their table structures.
- `outputs/guide_text_libraries.md`: Developer guide detailing text-based libraries and their structures.
- `docs/Wharton Research Data Services.csv`: General metadata and documentation regarding WRDS products and services.
