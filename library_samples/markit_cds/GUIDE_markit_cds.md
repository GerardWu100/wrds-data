# Markit CDS Guide

## Part 1: Conceptual Explanation

This folder stores the tiny Markit CDS samples. The folder now contains one
10-row CSV per live table in `markit_cds`, including the yearly CDS tables and
the smaller lookup tables.

Artifact details:

- File pattern: `*.csv`
- Source: `library_samples/export_small_samples.py`
- Coverage: 28 live WRDS tables in `markit_cds`
- Row rule: each CSV contains `limit 10` rows
- Column rule: each CSV uses `select *`, so it contains all columns from that
  table
